# WIP: Musical JoyCon

A fun way to play music on your Joy-Con

## How to run

1. Clone this project

```bash
git clone https://github.com/cpea2506/musical_joycon
```

2. Connect your Joy-Cons to your computer
3. Run the command below

```bash
cargo run -- <path to midi file>
```

4. Enjoy!
